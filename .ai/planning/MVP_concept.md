# Application - GymRatPlanner (MVP)

## Main problem
It is difficult to remember the results achieved during gym workouts. The application allows you to create workout templates consisting of a list of exercises and use them during subsequent workouts, where the new highest score achieved is saved in the workout template. This way

## Minimum set of features
- Saving, reading, viewing, and deleting workout templates
- Saving new personal bests for a given exercise
- Simple user account system to link users to their own workouts
- User profile page for saving workout preferences

## What is NOT included in the MVP
- Sharing training templates between accounts
- Analysis of results achieved by the user

## Success criteria
- 90% of users have created training templates with completed results
- 75% of users have at least 3 training templates
